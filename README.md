# NBA Player Performance Dashboard

## Overview
This project analyzes NBA player performance using Python and Tableau to identify trends in scoring efficiency and playing time.

## Tools Used
- Python (Pandas)
- Tableau
- Excel

## Data Processing
- Cleaned 16,000+ game-level records using Pandas
- Converted data types and handled missing values
- Created efficiency metrics:
  - Points per minute
  - Assists per minute
  - Rebounds per minute

## Analysis
- Aggregated player-level statistics
- Filtered out low-sample players for reliability
- Compared efficiency vs. minutes played

## Key Insight
Player efficiency generally increases with playing time, with elite players clustering in high-minutes, high-efficiency segments.
